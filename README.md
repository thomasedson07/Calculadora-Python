## Como executar 

Baixe os arquivos do repositório.

Abra o terminal.

Use o comando "cd (o caminho correspondente ao diretório onde os arquivos foram baixados).
Exemplo "cd /home/usuario/downloads".

Execute o script com ./calculadora.sh.
